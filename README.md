# 🎮 Games API - CRUD Avançado

API RESTful para gestão de um catálogo de jogos, construída em Node.js com Express e SQLite. Inclui autenticação JWT, relacionamentos entre tabelas e testes automatizados.

## 🛠️ Tecnologias
* **Runtime:** Node.js
* **Framework:** Express
* **Base de Dados:** SQLite
* **Segurança:** JSON Web Token (JWT) e BcryptJS
* **Testes:** Jest e Supertest
